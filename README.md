<h1> Тестовое задание на позицию Junior PHP-разработчик</h1>

## Быстрый старт

- Клонировать проект с github
- Выполнить команду `composer i`
- Скопировать файл `.env` и добавить в него данные для подключения к БД
- Выполнить команду `php artisan key:generate`
- Выполнить команду `php artisan migrate --seed`
- Выполнить команду `php artisan serve`
- Перейти по адресу, указанному в терминале
